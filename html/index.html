Live Github Actions Demo to App Studio

This application uses GitHub Actions with App Studio to get an app running in minutes.

Together, we're super awesome

Run an app using in a few easy steps

  1. Install the app into your fork of this repo buy using this gitops app https://actions.northdepot.ca/
  2. Install App-Studio Cluster Secrets as per the readme. This will be automated in future.
  3. Push changes to your repo and watch your app run in App Studio.
